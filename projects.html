AlgoPicks

  • Language(s): Swift, Python
  • Version: 1.1
  • Developers: Silas Nevstad

Description: AlgoPicks is a sports prediction and tracking app that uses machine learning algorithms to predict game outcomes. Utilizing the Firebase real-time database, the app allows users to track their bets in real-time and maintain a historical record of them, providing users with insights into their betting patterns and performance over time. With over 15,000 data points, AlgoPicks is able to predict game outcomes with a 70% accuracy rate in 2021. My goal with AlgoPicks is to create a valuable and reliable tool for both sports enthusiasts and those looking to improve their betting.

  • Language: Swift

Description: ChessClock is an app that I developed as a personal project, fueled by the frustration of not having a proper clock when playing chess with friends. I developed it in Swift in under 12 hours, and it enables chess players to play time-controlled games as well as setting custom time controls for each player and keeping track of the time remaining during the game. Its simple interface makes it easy to use for players in all age ranges.

AlgoTrader

  • Language: Python
  • Version: 1.2
  • Developers: Silas Nevstad

Description: The algorithm starts by selecting a certain number of stocks based on certain criteria, such as recent performance and overall financial health. Next, it performs a deeper analysis on a smaller subset of those stocks to determine which ones are the best fit for the portfolio. Once the final list of stocks is determined, the algorithm calculates the appropriate allocation for each stock in the portfolio. The algorithm then compares the current portfolio to the newly generated portfolio and makes trades to align them. The algorithm will also send a text message with the list of stocks that is bought or sold. Finally, it also calculates the balance changes and sends a text message with the list of stocks that were picked. The algorithm has been implemented on a Alpaca paper trading account and has resulted in an 18% increase in overall portfolio value, going from a starting balance of 100,000 to a current balance of 118,000.
